# Requisitos HTML y CSS

- Se deben usar etiquetas semánticas: <header>, <nav>, <main>, <section>, <article>, <footer>, <search>
- Utiliza grid layout o flexbox para el diseño de la página
- Utiliza media queries para ajustar el dispositivo en dispositivos móviles y tabletas
- Comprobar que el HTML es válido utilizando el validador W3C

# Estructura de directorios

El proyecto está organizado en carpetas y archivos de la siguiente manera:
mockup/
 - Contiene el ejemplo de landing page que quiero  Landingpage.png
 - Contiene el ejemplo de tienda que quiero  Shop.png

images/indexsimg
  - Contiene las imágenes utilizadas en el index.
styles/
  - Contiene los archivos CSS que definen el estilo del proyecto.
index.html
  - Archivo principal del proyecto que estructura el contenido y enlaza los estilos.
shop.html
  - Pagina que simula una tienda en vivo
